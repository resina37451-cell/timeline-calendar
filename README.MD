# Timeline Calendar (Obsidian Plugin)

A visual timeline calendar for Obsidian with time slots from **5:00 to 23:00**.

Instead of a traditional calendar, this plugin lets you plan your day using a **timeline view**, similar to Google Calendar but fully inside Obsidian.

---

## Features

- Timeline calendar (daily / weekly / monthly view)
- Time slots every 30 minutes
- Events with start and end time
- All-day events
- Event colors
- Notes inside events
- Mark events as completed
- Today indicator (current time line)
- Zoom levels:
  - Today
  - 1 week
  - 2 weeks
  - 1 month
  - 2 months
- Automatic language detection (Portuguese / English)

---

## Why this plugin?

Most calendar plugins for Obsidian focus only on dates.

This plugin focuses on **time planning**, which makes it perfect for:

- Students
- Daily planning
- Study routines
- Time blocking
- Productivity workflows

---

## Installation (Manual)

1. Download the latest release
2. Extract the folder
3. Copy the folder to: Vault/.obsidian/plugins/

4. Restart Obsidian
5. Enable **Timeline Calendar** in the community plugins settings

---

## How to use

After enabling the plugin:

1. Click the calendar icon in the left ribbon
2. The timeline view will open
3. Click on any time slot to create a new event
4. Click an event to edit it
5. Use the zoom buttons to switch between day, week, and month views
